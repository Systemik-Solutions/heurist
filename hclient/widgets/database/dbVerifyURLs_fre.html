Vérifiez les URL non valides dans les en-têtes d'enregistrement, les champs texte et fichier

La vitesse de vérification est limitée à 30 URL par minute pour éviter de bloquer le serveur. Nous recommandons de limiter la vérification à 150 vérifications (5 minutes), ce qui renverra une liste des URL incorrectes rencontrées jusqu'à ce point sous forme de fichier CSV et une recherche affichant les enregistrements les contenant afin que vous puissiez les corriger.

Vous pouvez ensuite exécuter à nouveau la fonction et elle revérifiera les URL incorrectes pour s'assurer qu'elles ont été corrigées, puis reprendra là où elle s'était arrêtée. Répétez aussi souvent que nécessaire pour corriger toute la base de données
